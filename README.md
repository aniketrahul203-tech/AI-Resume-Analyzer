# 🤖 AI Resume Analyzer

A web app that analyzes resumes against job descriptions using NLP techniques.

## 🚀 Features
- Upload PDF resume
- Extract text using PyPDF2
- Match with job description
- Show match score
- Identify missing skills
- Provide suggestions

## 🛠️ Tech Stack
- Python
- Streamlit
- PyPDF2
- NLP (regex + stopwords)

## ▶️ Run Locally
pip install streamlit PyPDF2  
streamlit run app.py
